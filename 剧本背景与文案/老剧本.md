# 《寂静校园：失踪档案》解谜游戏剧本

## 游戏概述
这是一款点击式网页解谜游戏，玩家通过在学校官网页面中寻找细微的异常和线索，逐步揭开学生失踪的真相。游戏共设7个场景，每个场景对应一个精心设计的网页界面。

## 核心机制
- **异常发现**：玩家需在看似正常的网页中找到不合逻辑、矛盾或隐藏的线索
- **线索收集**：每个场景提供关键信息碎片
- **逻辑推理**：将不同场景的线索联系起来，拼凑完整真相
- **多重结局**：根据发现的关键线索数量，解锁不同结局

## 人物设定
- **林小雨**：高二(3)班学生，三个月前失踪
- **刘天清**：高二(3)班学生，数学竞赛一等奖获得者，一个月前失踪
- **董新飞**：数学教师，刘天清的指导老师
- **王校长**：学校负责人
- **张保安**：学校保安

## 游戏剧本：七个场景

### 场景1：学校官网首页（已提供）

**异常点设计**：
1. **新闻日期异常**：数学竞赛新闻日期为“2024年10月15日”，但当前明显不是这个时间
2. **矛盾信息**：新闻中“刘天清荣获一等奖”与“学生失踪”主题形成暗黑对比
3. **隐藏链接**：“学校概况”链接指向`test1.html`（非标准命名）
4. **诡异细节**：网站使用大量红色主题，暗示危险/警告
5. **搜索功能**：输入特定关键词会触发隐藏反应

**玩家操作**：
- 点击数学竞赛新闻
- 点击“学校概况”链接
- 在搜索框输入“林小雨”或“失踪”

**隐藏线索**：
- 搜索“林小雨”后，页面会轻微抖动，右下角短暂出现“档案室密码：1024”文字
- 点击新闻日期会显示真实日期（比当前时间更早）

**跳转条件**：点击“学校概况”链接进入场景2

### 场景2：学校概况页面 (`test1.html`)

**页面设计**：
- 学校历史介绍
- 校园地图
- 领导团队展示
- 校训校徽说明

**异常点设计**：
1. **校园地图异常**：老校区区域被模糊处理，点击时提示“该区域暂不开放”
2. **领导照片**：王校长的照片比其他人的像素低，且眼神方向异常
3. **历史时间线**：1998年条目显示“旧实验楼拆除”，但无拆除原因
4. **校训变化**：当前校训是“厚德博学·追求卓越”，但早期校训被涂抹
5. **隐藏按钮**：连续点击校徽10次，出现“访问隐藏档案”按钮

**玩家操作**：
- 尝试点击老校区区域
- 仔细观察王校长照片
- 点击校徽多次
- 寻找“返回首页”链接以外的导航

**隐藏线索**：
- 王校长照片放大后，眼镜反射中有一个模糊的人影在旧实验楼窗口
- 使用场景1获得的密码“1024”可解锁隐藏档案页面

**跳转条件**：
- 成功进入隐藏档案页面，或
- 发现“图书馆新书通知”链接进入场景3

---

### 场景3：数字图书馆页面

**页面设计**：
- 图书检索系统
- 新书推荐
- 借阅排行榜
- 电子资源

**异常点设计**：
1. **借阅记录**：刘天清在失踪前一周密集借阅《都市传说研究》《异常心理学》《建筑结构安全》
2. **检索系统**：搜索“实验楼”显示“无相关结果”，但搜索“旧楼”有1条记录
3. **新书列表**：有《校园安全手册》《监控系统原理》等书，出版日期都在学生失踪后
4. **排行榜异常**：《沉默的证人》一书借阅量为0却排在榜首
5. **页面源码**：查看源码发现注释“<!--数据清理完成于2024-10-20-->”

**玩家操作**：
- 查看刘天清的借阅历史
- 尝试不同关键词搜索
- 点击《沉默的证人》书籍封面
- 查看页面源代码

**隐藏线索**：
- 《沉默的证人》封面点击后显示二维码，扫描进入“学生论坛”
- 源码中找到后台管理入口`admin.php`

**跳转条件**：
- 通过二维码进入场景4（学生论坛），或
- 通过“监控室申请”链接进入场景5

---

### 场景4：校园内部论坛（需要密码访问）

**页面设计**：
- 匿名发帖区
- 各班级讨论版
- 校园活动板块
- 失物招领

**异常点设计**：
1. **大量删帖**：高二(3)班板块近三个月帖子数量异常少
2. **匿名爆料**：用户“知情人”发帖“旧实验楼晚上有灯光”，帖子发布时间为凌晨2:15
3. **被屏蔽内容**：某些帖子只显示标题，内容为“该帖已被管理员屏蔽”
4. **用户状态**：刘天清账号最后登录时间显示为“在线”，但实际已失踪
5. **私信功能**：收到来自“小雨不哭”的未读消息（需密码解锁）

**玩家操作**：
- 查看高二(3)班板块
- 阅读“知情人”帖子
- 尝试回复被屏蔽的帖子
- 猜测“小雨不哭”的账号密码

**隐藏线索**：
- “小雨不哭”账号密码为林小雨的生日（可从场景1的新闻中推断）
- 私信中包含旧实验楼的平面图手绘照片
- “知情人”IP地址显示为教师办公室区域

**跳转条件**：
- 通过私信中的线索进入场景6（旧实验楼），或
- 通过论坛“校务公开”链接进入场景5

---

### 场景5：监控室/安保系统页面

**页面设计**：
- 监控区域选择
- 录像回放系统
- 安保日志
- 访客登记

**异常点设计**：
1. **监控盲区**：校园地图显示旧实验楼区域无监控覆盖
2. **录像缺失**：林小雨失踪当天（9月12日）的监控录像显示“文件损坏”
3. **安保日志**：张保安在9月12日日志中记录“巡查时听到旧楼有声响，检查无异常”
4. **时间矛盾**：刘天清失踪当天，监控显示他进入图书馆但未离开，但图书馆闭馆记录有他签出
5. **访客记录**：董新飞老师频繁在非工作时间登记进入旧实验楼

**玩家操作**：
- 尝试修复损坏的录像文件
- 对比不同日期的安保日志
- 查看董新飞的访客记录详情
- 寻找监控系统后门

**隐藏线索**：
- 修复录像需要密码，可从场景3图书馆源码中获得
- 修复后的录像显示林小雨被一个穿校服的人引向旧实验楼
- 访客记录中有多次“设备维修”备注，但维修公司不存在

**跳转条件**：
- 发现旧实验楼入口密码线索，进入场景6

---

### 场景6：旧实验楼内部管理系统

**页面设计**：
- 楼宇结构图
- 实验室使用登记
- 设备清单
- 安全规程

**异常点设计**：
1. **使用记录**：B203实验室连续三个月每晚都有使用记录，使用人为“董新飞”
2. **设备异常**：设备清单中有“脑电波监测仪”“记忆干预设备”等非常规教学设备
3. **结构矛盾**：平面图显示有地下层，但官方文件称无地下室
4. **访问权限**：只有王校长和董新飞有B203实验室的永久权限
5. **安全警报**：系统日志显示9月12日、10月15日有安全警报，但被手动关闭

**玩家操作**：
- 查看B203实验室的详细记录
- 对比设备清单与采购记录
- 寻找进入地下层的方法
- 破解实验室门禁密码

**隐藏线索**：
- 实验室密码为刘天清的学号（可从场景1新闻图片中获取）
- 地下层入口隐藏在储藏室，需要物理钥匙
- 系统中发现“记忆提取实验”项目文件

**跳转条件**：
- 成功进入B203实验室监控，进入最终场景7

### 场景7：真相揭示页面（多重结局）

**页面设计**：
根据玩家发现的线索数量，呈现不同版本的真相：

**结局A：完美真相（发现所有线索）**
- 完整视频：董新飞进行非法记忆实验，试图提取数学天才的思维模式
- 王校长为学校声誉掩盖真相
- 林小雨偶然发现实验，被迫参与
- 刘天清主动调查好友失踪，也被卷入
- 两人目前被囚禁在地下室，记忆被干预
- 提供报警证据包和具体位置

**结局B：部分真相（发现70%线索）**
- 知道董新飞进行非法实验
- 知道两名学生被囚禁
- 但不知道具体位置和完整动机
- 可选择报警，但证据不足

**结局C：表面真相（发现40%线索）**
- 只知道有异常，怀疑董新飞
- 但没有决定性证据
- 游戏以开放式悬念结束

**结局D：被误导（点击错误线索）**
- 被引导相信是学生自己离家出走
- 或相信是外部人员作案
- 游戏结束，真相被掩盖

## 技术实现要点

1. **异常注入**：在原HTML基础上添加隐藏元素、矛盾数据、非常规交互
2. **状态追踪**：使用localStorage记录玩家发现的线索和进度
3. **动态响应**：根据玩家行为改变页面内容（如搜索关键词触发不同反应）
4. **密码系统**：线索之间相互关联，形成解谜链条
5. **时间机制**：某些线索只在特定操作顺序后出现
6. **反侦察设计**：NPC（页面元素）会对玩家的异常调查有“反应”

## 叙事主题
- 表面光鲜与隐藏黑暗的对比
- 权威叙事与个人真相的冲突
- 技术伦理与教育异化
- 记忆、身份与自我

## 玩家体验设计
1. **初期**：正常浏览，偶然发现小异常
2. **中期**：主动寻找矛盾，建立假设
3. **后期**：串联线索，推理完整故事
4. **终局**：做出选择，面对后果

这个剧本充分利用网页元素作为叙事媒介，将学校官网转化为侦探现场，让玩家在点击、滚动、输入中体验调查过程，最终揭露隐藏在代码和数据背后的黑暗真相。
